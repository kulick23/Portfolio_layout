Что я использую в работе

HTML5

Именно он создает каркас вашего сайта или приложения, а затем верстка позволяет мне создать более SEO-оптимизированную структуру вашего продукта.

CSS3

Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!

JavaScript

Этот язык программирования позволяет оживить все: от слайдеров, форм, подсказок, всплывающих окон до серверов и многого другого.

jQuery

Библиотека jQuery позволяет ускорить разработку. Без необходимости интегрировать в проект её мы не будем, но навык работы с ней присутствует.

React

Эта библиотека позволяет создавать web-приложения. Мы можем создать максимально интерактивный продукт именно под ваш случай.

Node.js

Эта платформа позволяет создать сервер для вашего проекта – "мозг", который будет выполнять действия, которые пользователь не видит.

MongoDB

Это нереляционная база данных, которая будет хранить данные вашего сайта или приложения.
